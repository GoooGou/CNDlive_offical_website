# 🚀 Markdown 极速入门：10 分钟搞定文档写作

> 目标：读完本文，你能独立写出排版清爽、结构清晰的 **README / 博客 / 接口文档**。

---

## 1. 什么是 Markdown？

| 特点 | 说明 |
|---|---|
| 纯文本 | 用任何编辑器都能打开，不担心格式乱码 |
| 语法简单 | 记住 10 个符号就能写 90% 的内容 |
| 一键变美 | 渲染后自动得到 HTML / PDF / 网站 |

---

## 2. 标题与段落

```markdown
# 一级标题（通常只出现一次）
## 二级标题
### 三级标题
```

段落之间**空一行**即可自动换行，无需 `<br/>`。

---

## 3. 文本样式（4 个就够）

| 效果 | 语法 | 示例 |
|---|---|---|
| 粗体 | `**文字**` | **重要** |
| 斜体 | `*文字*` | *引用* |
| 行内代码 | `` `代码` `` | `print()` |
| 删除线 | `~~文字~~` | ~~过时~~ |

---

## 4. 列表（有序 & 无序）

无序：
```markdown
- 苹果
- 香蕉
  - 青香蕉   ← 缩进 2 空格
```

有序：
```markdown
1. 打开冰箱
2. 把大象塞进去
3. 关门
```

---

## 5. 链接 & 图片

```markdown
[百度一下](https://www.baidu.com)

![alt 文字](https://example.com/cat.png)
```

> 内部文章用「相对路径」：`[快速跳转](./syntax.md)`

---

## 6. 代码块（高亮 + 复制）

````markdown
```javascript
function hello() {
  console.log('Hello Markdown!');
}
```
````

渲染后：

```javascript
function hello() {
  console.log('Hello Markdown!');
}
```

---

## 7. 引用 & 分隔线

引用：
```markdown
> 这是一个引用块，可嵌套
>> 嵌套引用
```

分隔线：
```markdown
---
```

---

## 8. 表格（对齐一目了然）

```markdown
| 姓名 | 年龄 | 城市 |
| :--- | :-: | ---: |
| 张三 | 28 | 北京 |
| 李四 | 25 | 上海 |
```

- `:---` 左对齐，`:--:` 居中，`---:` 右对齐

---

## 9. 任务清单（GitHub 风格）

```markdown
- [x] 写完需求
- [ ] 单元测试
- [ ] 发布上线
```

效果：
- [x] 写完需求
- [ ] 单元测试
- [ ] 发布上线

---

## 10.  Emoji 与转义

| 用法 | 示例 |
|---|---|
| 直接输入 | `:smile:` → 😄 |
| 转义符号 | `\*不是斜体\*` → *不是斜体* |

---

## 11. 30 秒排版口诀

> `#` 标题  
> `*` 强调  
> `-` 列表  
> `` ` `` 代码  
> `>` 引用  
> 空行分段，**搞定**！
